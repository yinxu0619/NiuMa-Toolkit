# 牛马工具箱 · NiuMa-Toolkit

> **钱是公司发的，命是自己的；数据是本地的，班是上不完的。**  
> 一款**零登录、无广告、全本地**的打工人app：摸鱼按秒算钱、薅羊毛有据可查、今日结余一眼看懂、退休倒计时替你数日子。  
> 不上传你的薪资和摸鱼记录——老板不配拥有这份 JSON。

**唯一会联网**：设置里你**亲手点开**「获取假日安排」时，从 [holiday-cn](https://github.com/NateScarlet/holiday-cn) 把国务院那套放假/调休表同步到手机，**存完就断网，要不谁知道假日办的老爷们打算怎么编排牛马呢**。  
毕竟调休这事，连日历都看不懂，牛马只想要一个准话：明天到底算不算人过的日子。

**开源协议**：[MIT](LICENSE) — fork 随意，改完记得自己爽。

## 功能概览

| Tab | 你能干啥 |
|-----|----------|
| **今日** | 距午休/下班**秒级**倒计时；已赚按秒滚（像真在从老板兜里掏钱）；摸鱼+薅羊毛 vs 支出、净结余；房贷/通勤等固定成本进度条；**退休倒计时**（再熬熬就自由）；下一法定休；日历里标的**年假/病假**会当成带薪爽摸日，**通勤按 0 算**，状态和结余跟日历对齐。 |
| **摸一把** | 中间那颗猫爪：带薪如厕 / 开摆 / 无效会议……**毫秒计时、按秒结算**，停表还有结算骚话；不选类型也能开摸，停时没选类型就不落库（给选择困难症留条活路）。 |
| **统计** | 今日净结余、已薅明细可展开；本周/本月汇总；分类与时间线，删记录也有随机悼词。 |
| **羊毛工具箱** | 咖啡零食饮料充电、通勤、午饭、万能支出、假期、日历……**今日已薅金额 + 热量 + 文案**，把白嫖量化成尊严。 |
| **设置** | 时间/薪资/假期/个人；**法定假日表**（可选同步）；数据导出与清空；**统一浅色界面**（不费眼、不整花里胡哨主题切换）。 |

### 配置怎么影响你「算得爽不爽」

- **作息**：上下班、午休开关 → 有效工时、时薪/秒薪、倒计时逻辑。
- **薪资**：月薪 ÷ 计薪天数 → 全 App 共用一把尺子量老板。
- **假期**：年假/病假额度在设置里填，**日历/假期管理里标记哪一天**会扣额度；带薪假日在首页按「爽摸」算钱，病假走养伤话术，年假走薅假话术——**都是正经权益，别替公司省**。
- **个人**：生日 → 退休倒计时（默认 60 岁可改，每秒都在提醒你人生苦短）。
- **房贷 + 通勤**：摊到工作日；今日固定支出里一起算，进度条上的骚话会告诉你「还清没、还得干多久」。

### 骚话从哪来

下班打卡、结余正负、固定费用、删除记录、会议结算……大量**随机骚话**都在 `constants/coverageCopy.ts`、`constants/copy.ts` 等，**纯本地生成**，没有 AI 偷看你的羞耻心。

## 法定假日（可选联网）

- **默认**：内置兜底日期，给「下一个法定休」一个大概盼头，**不联网**。
- **开启同步后**：拉 isOffDay 表（含调休），**写进本地**，之后**不再自动请求**。今日若在表里 → 首页直接「法定休息、爽摸计价」。  
→ **全 App 只有这一处会访问网络**，且必须你主动操作，不搞后台小动作。

## 如何运行

```bash
npm install
npm start
```

Expo Go 扫码真机，或 `npm run ios` / `npm run android`。图标与闪屏在 `assets/`。

### iOS 模拟器（Mac + Xcode）

```bash
npm install
npm run ios
```

若已 `npm start`，终端按 **`i`** 也行。

```bash
npm run build:android:apk   # 侧载 APK
```

## 目录结构（迷路时看一眼）

底部 **5 Tab**：**今日** · **统计** · **摸一把（猫爪 C 位）** · **羊毛工具箱** · **设置**。

```
├── app/(tabs)/
│   ├── home.tsx          # 今日：倒计时、结余、请假/假日联动
│   ├── reports.tsx       # 统计
│   ├── _layout.tsx       # Tab + 中央摸一把入口
│   ├── moyu.tsx          # 摸一把主流程
│   ├── more.tsx          # 工具箱聚合
│   ├── expense, commute, leave, calendar, charge, lunch, ...
│   └── settings.tsx
├── components/
├── constants/            # 配色、骚话、成就文案
├── contexts/             # SalaryContext, ToastContext
├── lib/
│   ├── storage.ts
│   ├── holidays.ts
│   ├── salary.ts, workTime.ts, mortgage.ts, retirement.ts, today.ts, ...
└── types/
```
