# 牛马工具箱 · NiuMa-Toolkit

> **钱是公司发的，命是自己的；数据是本地的，班是上不完的。**  
> 一款**零登录、无广告、全本地**的打工人app：摸鱼按秒算钱、薅羊毛有据可查、今日结余一眼看懂、退休倒计时替你数日子、FIRE 速算辞职成本。  
> 不上传你的薪资和摸鱼记录——老板不配拥有这份 JSON。

**开源协议**：[MIT](LICENSE) — fork 随意，改完记得自己爽。

## 功能概览

| Tab | 你能干啥 |
|-----|----------|
| **今日** | 距午休/下班**秒级**倒计时；已赚按秒滚（像真在从老板兜里掏钱）；摸鱼+薅羊毛 vs 支出、净结余；房贷/通勤/社保等固定成本进度条；**退休倒计时**（再熬熬就自由）；下一法定休；日历里标的**年假/病假**会当成带薪爽摸日，**通勤按 0 算**，状态和结余跟日历对齐。 |
| **摸一把** | 中间那颗猫爪：带薪如厕 / 开摆 / 无效会议……**毫秒计时、按秒结算**，停表还有结算骚话；不选类型也能开摸，停时没选类型就不落库（给选择困难症留条活路）。 |
| **统计** | 今日净结余、已薅明细可展开；本周/本月汇总；分类与时间线，删记录也有随机悼词。 |
| **羊毛工具箱** | 咖啡零食饮料充电、通勤、午饭、万能支出、假期、日历、🔥 **FIRE 退休计算器**……**今日已薅金额 + 热量 + 文案**，把白嫖量化成尊严。 |
| **设置** | 时间/薪资/假期/个人；**法定假日表**（可选同步）；数据导出与清空；**统一浅色界面**（不费眼、不整花里胡哨主题切换）。 |

## 🔥 FIRE 提前退休计算器

### 「老子不干了！」速算

页面顶部一键速算：**辞职后以灵活就业身份、最低 60% 基数缴养老**，自选职工医保或居民医保，秒出每月/每天自缴成本 + 退休能拿多少。

### 养老保险（全国统一公式，无省份差异）

**① 加权平均缴费指数**（解决「历史 3 倍 + 未来 60%」混合基数问题）

$$
\text{加权指数} = \frac{\text{已缴年限} \times \text{历史基数比例} + \text{未来年限} \times \text{未来基数比例}}{\text{已缴年限} + \text{未来年限}}
$$

> 基数比例：60% → 0.6，300% → 3.0

**② 基础养老金（月）**

$$
\text{基础养老金} = \frac{\text{月社平} \times (1 + \text{加权指数})}{2} \times \text{总缴费年限} \times 1\%
$$

**③ 个人账户养老金（月）**

$$
\text{未来个账入账} = \text{月社平} \times \text{未来基数} \times 8\% \times \text{未来缴费月数}
$$

$$
\text{个账养老金} = \frac{\text{已缴余额} + \text{未来入账}}{139}
$$

> 139 为 60 岁退休计发月数（国家固定值）

**④ 每月养老缴费（纯养老，不含医保）**

| 身份 | 公式 | 说明 |
|------|------|------|
| 在职职工 | 月社平 × 基数 × **8%** | 个人部分，单位另缴 16% |
| 灵活就业 | 月社平 × 基数 × **20%** | 全部自缴，无单位 |

**⑤ 退休月养老金 = 基础养老金 + 个账养老金**

### 医疗保险（各省系数库自动匹配）

| 类型 | 公式 | 备注 |
|------|------|------|
| 职工医保（在职） | 月社平 × 基数 × **2%** | 个人部分 |
| 灵活就业职工医保 | 月社平 × 基数 × **10%** | 全自缴；北京固定 100% 社平基数 |
| 城乡居民医保 | **年缴** ÷ 12 | 默认 380 元/年，终身缴费 |

内置全国核心省市系数（退休免缴年限、基数限制等），支持在线更新。

### 分段缴费

支持设定多段未来缴费基数（如前 5 年 100%、后续 60%），加权计算养老和医保。

### FIRE 停缴（未来基数 = 0%）

未来基数设为 0% → 未来养老/医保缴费全部归零，退休金仅按历史缴费计算。

## 联网说明

**默认不联网**，所有数据手动填写、本地存储。以下 3 处**仅在你主动点击按钮时**才会联网：

| 功能 | 触发方式 | 数据来源 |
|------|---------|----------|
| **同步法定假日** | 设置页 →「同步节假日（含调休）」 | [holiday-cn](https://github.com/NateScarlet/holiday-cn)（国务院假日安排） |
| **获取社平工资** | FIRE 页 →「获取社平工资」 | 本项目 `data/social_avg_salary.json` |
| **更新医保系数** | FIRE 页 →「更新医保系数库」 | 本项目 `data/medical_region_coefficients.json`（GitHub Raw + jsDelivr CDN 双通道） |

联网失败自动使用 App 内置兜底数据，**不影响使用**。无定位、无后台请求、无数据上传。

## 配置怎么影响你「算得爽不爽」

- **作息**：上下班、午休开关 → 有效工时、时薪/秒薪、倒计时逻辑。
- **薪资**：月薪 ÷ 计薪天数 → 全 App 共用一把尺子量老板。
- **假期**：年假/病假额度在设置里填，**日历/假期管理里标记哪一天**会扣额度；带薪假日在首页按「爽摸」算钱，病假走养伤话术，年假走薅假话术——**都是正经权益，别替公司省**。
- **个人**：生日 → 退休倒计时（默认 60 岁可改，每秒都在提醒你人生苦短）。
- **房贷 + 通勤**：摊到工作日；今日固定支出里一起算，进度条上的骚话会告诉你「还清没、还得干多久」。
- **FIRE 社保 / 医保**：每日社保、医保成本同步到首页进度条固定支出，修改任意参数实时刷新。

### 骚话从哪来

下班打卡、结余正负、固定费用、删除记录、会议结算……大量**随机骚话**都在 `constants/coverageCopy.ts`、`constants/copy.ts` 等，**纯本地生成**，没有 AI 偷看你的羞耻心。
